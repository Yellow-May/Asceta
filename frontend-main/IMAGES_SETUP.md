# Images Setup

The images folder should be copied to the `public` directory so that images can be accessed via `/images/` path.

## Setup Instructions

1. Copy the `images` folder from the project root to `frontend-main/public/`
2. The folder structure should be: `frontend-main/public/images/`
3. All images will be accessible at `/images/filename.ext`
