# ASCETA Main Frontend

Main frontend application for ASCETA website (public pages + student portal).

## Setup

1. Install dependencies:
```bash
yarn install
```

2. Create a `.env` file:
```bash
cp .env.example .env
```

3. Update the `.env` file with your backend API URL.

4. Start the development server:
```bash
yarn dev
```

The application will be available at `http://localhost:3000`

## Features

- Public website pages (Home, About, Admission, Academics, News, Events)
- Student portal with authentication
- Responsive design
- News and Events integration with backend API
